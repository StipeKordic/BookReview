# BookReviewer Project

This project allows users to register on BookReviewer and contribute their own book reviews. Users can also edit or delete their submitted reviews at any time. Additionally, they can browse reviews from other users to help them decide whether a particular book is worth reading.
## Features
- User registration
- Adding reviews
- Editing reviews
- Deleting reviews
- Browsing other reviews

## Local Setup

#### 1. Clone repo on your machine
  ```bash
  git clone https://github.com/StipeKordic/BookReview.git
```

#### 2. Create docker containers
Move to the project directory and run the following command in the terminal:
  ```bash
  docker-compose up -d
```
